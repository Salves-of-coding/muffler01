# Chapter 2. 변수와 자료형

## 변수의 이해
- print() 함수 : () 안의 내용을 출력
- name = value : name이라는 변수에 value라는 값을 넣는다.
- 변수명 선언 규칙
1. 알파벳, 숫자, 밑줄로 선언할 수 있다.
1. 변수명은 의미 있는 단어로 표기하는 것이 좋다.
1. 변수명은 대소문자가 구분된다.
1. 특별한 의미가 있는 예약어는 사용할 수 없다.
## 자료형과 기본 연산
- 기본 자료형
1. 정수형 : 양수, 0, 음수
  - 선언 형태 : data = 1
1. 실수형 : 소수점이 포함된 실수
  - 선언 형태 : data = 9.0
1. 문자형 : 따옴표에 들어가 있는 문자
  - 선언 형태 : data = 'abc'
1. 불린형 : 참 또는 거짓 
  - 선언 형태 : data = True
